# AGENTS.md

## 项目规则

1. 本项目是个人学习管理应用，不是普通任务管理器。
2. 所有功能必须围绕 INTP 问题驱动学习法。
3. 修改代码时必须保持中文界面。
4. 不要破坏 SQLite 数据结构。
5. 新增功能前先考虑是否服务于：
   - 问题驱动
   - 闭卷回忆
   - 错因分析
   - 间隔复习
   - 主线 / 插问机制
6. 插问功能必须始终绑定主线锚点。
7. 不允许把插问变成无限扩展的资料搜索。
8. 复习功能必须使用 1-3-7-14 规则。
9. 掌握度低于 70% 的知识点必须进入重点复习区。
10. 完成修改后要说明：
    - 改了哪些文件
    - 如何运行
    - 如何验证功能

## 本地环境

- 使用 MSYS/MSYS2 编译 C 程序。
- 使用 Git Bash 处理 GitHub 登录和认证相关操作。
